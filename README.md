## 📖 Descripción del Proyecto

Aplicación de consola desarrollada en Java para gestionar una biblioteca universitaria.  
Permite registrar libros, estudiantes y préstamos, incorporando validaciones y manejo de errores mediante excepciones personalizadas.

---

## 🛠️ Tecnologías utilizadas

- Lenguaje: Java 23 (JDK 23) 
- Gestor de Dependencias: Apache Maven  
- IDE: Visual Studio Code  
- Modelado: UML (Diagrama de Clases)

---

## ⚙️ Funcionalidades

- Registro de libros y estudiantes  
- Gestión de préstamos  
- Validaciones de disponibilidad  
- Manejo de excepciones personalizadas  
- Límite de préstamos por estudiante  
- Búsqueda de libros por título  
- Listado de préstamos por estudiante  
- Cálculo de multa mediante recursividad  

---

## 📂 Estructura del proyecto

- `model` → clases principales (Libro, Estudiante, Prestamo)  
- `service` → lógica de negocio  
- `exception` → excepciones personalizadas  
- `ui` → clase Main (interfaz de ejecución)
